# Product Dashboard (JavaScript)

## Overview
This project is a responsive product dashboard built using HTML, CSS, and JavaScript. It dynamically loads and displays product data, simulating a real-world e-commerce interface.

## Features
- Fetches and displays product data from an API
- Dynamically renders products using JavaScript
- Responsive layout for different screen sizes
- Organized structure with separate HTML, CSS, and JS files

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- External API (product data)

## How to Run
1. Open the project folder
2. Launch `index.html` in your browser
3. View the product dashboard interface
